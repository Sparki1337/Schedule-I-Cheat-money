Money Patcher для Schedule I

Удобный инструмент для изменения баланса денег в игре

Возможности программы

  • Изменение количества денег на счету
  • Изменение количества наличных денег
  • Просмотр текущего баланса во всех сохранениях
  • Автоматический запуск игры после изменений
  • Создание резервных копий сохранений
  • Загрузка сохранений из резервных копий
  • Поддержка русского и английского языков

Как использовать

  1. Скачайте архив с программой
  2. Распакуйте архив в любую папку
  3. Запустите файл money_patcher.py двойным кликом или из командной строки
  4. Следуйте инструкциям в программе

Примечание: Для работы программы требуется Python 3.6 или выше. Если у вас не установлен Python, вы можете скачать его здесь.
